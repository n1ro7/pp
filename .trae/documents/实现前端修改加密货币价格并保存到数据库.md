# 实现前端修改加密货币价格并保存到数据库

## 1. 后端实现

### 1.1 修改 CryptocurrencyController
- 添加一个 PUT 方法 `/api/crypto/prices/{symbol}` 用于更新单个加密货币的价格
- 添加一个 PUT 方法 `/api/crypto/prices/batch` 用于批量更新加密货币的价格

### 1.2 修改 CryptocurrencyService
- 添加 `updateCryptocurrencyPrice` 方法用于更新单个加密货币的价格
- 添加 `batchUpdateCryptocurrencyPrices` 方法用于批量更新加密货币的价格

### 1.3 修改 CryptocurrencyRepository
- 已经有 `findBySymbol` 和 `save` 方法，不需要额外修改

## 2. 前端实现

### 2.1 修改 Admin 页面
- 添加一个新的标签页 "价格管理"，用于管理加密货币价格
- 实现加密货币价格列表展示
- 添加编辑价格功能，包括单个编辑和批量编辑
- 添加保存按钮，将修改后的价格提交到后端

### 2.2 修改 cryptoPriceService.js
- 添加 `updateCryptocurrencyPrice` 方法用于调用后端API更新单个价格
- 添加 `batchUpdateCryptocurrencyPrices` 方法用于调用后端API批量更新价格

### 2.3 修改 PriceContext.jsx
- 确保价格数据能实时更新，当价格被修改后能立即反映在前端

## 3. 实现步骤

1. 首先修改后端代码，添加价格更新API
2. 然后修改前端Admin页面，添加价格管理功能
3. 最后测试整个流程，确保前端修改价格后，数据库的数据能随之改变

## 4. 预期效果

- 管理员可以在Admin页面的价格管理标签页中查看所有加密货币的价格
- 管理员可以编辑单个或多个加密货币的价格
- 编辑后点击保存按钮，修改后的价格会保存到数据库
- 其他用户刷新页面后能看到更新后的价格
- PriceContext中的价格数据会自动更新，所有使用价格数据的组件都会实时反映最新价格
